# SQL Data Analysis Projects

## Project Description
This repository contains SQL-based analytical tasks focused on user activity, sales performance, and audiobook listening behavior.

## Tasks Performed

### User Activity Analysis
- Count unique users
- Identify top users
- Calculate average metrics
- User lifetime calculation

### Sales & Employees Analysis
- Product category analysis
- Sales aggregation
- Employee performance
- Join tables

### Audiobooks Analysis
- Listening behavior
- Popular books
- Completion rate
- Users segmentation

## Tools Used
- SQL
- MySQL
- PostgreSQL

## Skills Demonstrated
- JOIN and LEFT JOIN
- GROUP BY and HAVING
- Aggregations
- Filtering
- Business logic implementation

## Project Result
- Multiple business problems solved
- Data aggregated and analyzed
- Insights extracted using SQL
