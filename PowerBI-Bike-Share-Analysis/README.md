# Bike Share Analytics Dashboard

## Project Description
This project analyzes bike-sharing data in the San Francisco Bay Area using Power BI. The dashboard provides insights into trip patterns, user behavior, and station performance.

## Dataset
San Francisco Bike Share (Google BigQuery public dataset)

## Business Questions
- How trips change over time
- Peak usage days and hours
- Most popular stations
- Customer vs Subscriber comparison
- Most popular routes

## Dashboard Features
- Time series analysis
- Trips volume metrics
- Peak hours analysis
- Geographic visualization
- User segmentation
- Station performance

## Tools Used
- Power BI
- Data modeling
- DAX
- Interactive filters

## Skills Demonstrated
- Dashboard design
- Data modeling
- Time analysis
- Business insights
- Interactive reporting

## Project Result
- Identified peak trip periods
- Compared user segments
- Visualized geographic data
- Created interactive dashboard
