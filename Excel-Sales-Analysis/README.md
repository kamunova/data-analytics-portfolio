
# Excel Sales Analysis

This project focuses on analyzing sales data using Microsoft Excel. The goal was to clean the data, perform calculations, and create visualizations to identify key insights.

- Using Excel formulas
- Sales analysis
- Creating charts
- Building summary tables

- Charts
- Formulas (SUM, IF, COUNT, etc.)

## Skills Demonstrated
- Data analysis
- Data cleaning
- Visualization
- Business insights

## Project Result
- Sales data analyzed
- Charts created
- Key metrics calculated
- Summary report prepared
